Semantic Elements

Semantic HTML elements are tags that provide meaning and context to the content on a webpage. Unlike non-semantic tags like <div> and <span>, which are purely for styling, semantic tags like <header>, <nav>, and <article> describe the content they contain. This helps search engines, screen readers, and other machines understand the structure of your page.

Why Use Semantic Elements?

Using semantic elements improves accessibility, SEO, and code readability. Screen readers can use them to navigate the page more effectively, and search engines can better understand the importance of different sections. For developers, semantic tags make the code easier to read and maintain.

Example: Semantic Layout

<header>
  <h1>My Website</h1>
  <nav>
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </nav>
</header>

<main>
  <article>
    <h2>Article Title</h2>
    <p>This is the main content of the article.</p>
  </article>
</main>

<footer>
  <p>© 2025 My Website</p>
</footer>
