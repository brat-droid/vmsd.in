Box Model

The CSS box model is a fundamental concept that describes how elements are rendered on a webpage. This article covers the different parts of the box model and how they work together to create the layout of a page.

Content, Padding, Border, and Margin

The box model is composed of four parts: the content, padding, border, and margin. The content is the actual content of the element, such as text or an image. The padding is the space between the content and the border. The border is a line that goes around the padding and content. The margin is the space between the border and other elements on the page.

Example: Box Model Properties

.box {
  width: 100px;
  height: 100px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}

Box-Sizing

The "box-sizing" property allows you to change how the box model is calculated. By default, the width and height of an element are calculated based on the content, padding, and border. The "box-sizing" property can be used to change this so that the width and height are calculated based on the content only. This can be useful for creating more predictable layouts.

Example: box-sizing: border-box

.box-sizing-example {
  box-sizing: border-box;
  width: 100px;
  padding: 20px;
  border: 5px solid red;
}
