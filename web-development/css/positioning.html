Positioning

CSS positioning allows you to control the position of elements on a webpage. This article covers the different positioning properties and how to use them to create complex layouts.

Static, Relative, and Absolute Positioning

There are several different positioning properties in CSS, including static, relative, and absolute. This article will cover the differences between these properties and when to use each.

Example: Absolute Positioning

.parent {
  position: relative;
}

.child {
  position: absolute;
  top: 10px;
  left: 10px;
}

Fixed and Sticky Positioning

Fixed and sticky positioning are two other positioning properties that can be used to create interesting effects. This article will cover how to use these properties to create elements that are fixed to the viewport or that stick to the top of the page as the user scrolls.

Example: Fixed Positioning

.fixed-header {
  position: fixed;
  top: 0;
  width: 100%;
}
