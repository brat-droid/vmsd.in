Basic CSS

CSS (Cascading Style Sheets) is the language used to style the visual presentation of a webpage. It allows you to control the colors, fonts, spacing, and layout of your content. This article covers the fundamental concepts you need to get started with CSS.

Selectors and Properties

CSS works by selecting HTML elements and applying styles to them. This is done using selectors, which target elements based on their tag name, class, or ID. Once an element is selected, you can apply a variety of properties to change its appearance.

Example: Basic CSS Rule

p {
  color: blue;
  font-size: 16px;
}

The Cascade

The "cascading" in CSS refers to the rules that determine how styles are applied when multiple rules target the same element. Understanding the cascade is essential for writing predictable and maintainable CSS. This article will cover the basics of specificity and inheritance.
