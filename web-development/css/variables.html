Variables

CSS variables, also known as custom properties, are a powerful feature that allow you to store values for reuse throughout your stylesheet. This article covers how to use CSS variables to create more maintainable and flexible designs.

Declaring and Using Variables

This article will cover how to declare CSS variables using the -- syntax and how to use them using the var() function. We will also cover how to set a fallback value for a variable in case it is not defined.

Example: Declaring and Using a CSS Variable

:root {
  --primary-color: #007bff;
}

.button {
  background-color: var(--primary-color);
}

Scope and Inheritance

CSS variables are scoped to the element they are declared on and are inherited by their descendants. This article will cover how to use this to your advantage to create more flexible and modular designs.
