Absolute and Relative Units

In CSS, units are used to specify the size of elements and their properties. This article covers the difference between absolute and relative units, and when to use each.

Absolute Units

Absolute units are fixed and do not change based on the size of the screen or the parent element. The most common absolute unit is the pixel (px). Absolute units are useful when you need to specify a precise size, but they can be problematic for responsive design.

Example: Absolute Unit (px)

.box {
  width: 200px;
  height: 100px;
}

Relative Units

Relative units are flexible and change based on the size of the screen or the parent element. Common relative units include em, rem, and %. Relative units are essential for creating responsive designs that work well on a variety of devices.

Example: Relative Unit (em)

.text {
  font-size: 1.2em; /* 1.2 times the parent's font-size */
}
