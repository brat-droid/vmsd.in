JavaScript Fundamentals Review

This article provides a review of the fundamental concepts in JavaScript. It is intended for those who have some experience with JavaScript and want to refresh their knowledge.

Data Types and Variables

This section will review the different data types in JavaScript, including strings, numbers, booleans, and objects. We will also review how to declare and use variables.

Example: Data Types

let str = "Hello";       // String
let num = 123;         // Number
let bool = true;       // Boolean
let obj = {};          // Object
let arr = [];          // Array
let undef = undefined; // Undefined
let nul = null;        // Null

Control Flow

This section will review the different control flow statements in JavaScript, including if/else statements, for loops, and while loops. We will also review how to use functions to control the flow of a program.

Example: If/Else Statement

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
