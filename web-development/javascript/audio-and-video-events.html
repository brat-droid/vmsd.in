Audio and Video Events

The HTML5 audio and video elements provide a number of events that can be used to control playback and respond to changes in the media. This article covers how to use these events in JavaScript.

Media Events

This article will cover the different media events that are available, such as the play, pause, and ended events. We will also cover how to use these events to create a custom media player.

Example: Play/Pause Event

const video = document.getElementById("myVideo");
video.addEventListener("play", () => {
  console.log("Video started playing.");
});
video.addEventListener("pause", () => {
  console.log("Video paused.");
});

Track Events

In addition to the media events, there are also a number of track events that can be used to respond to changes in the text tracks of a media element. This article will cover how to use these events to create a more accessible media player.
