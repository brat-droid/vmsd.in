Booleans and Numbers

Booleans and numbers are two of the primitive data types in JavaScript. This article covers how to use booleans and numbers in your code.

Booleans

A boolean is a data type that can have one of two values: true or false. Booleans are often used in conditional statements to control the flow of a program. This article will cover how to use booleans in your code and how to perform logical operations on them.

Example: Boolean Logic

let isActive = true;
if (isActive) {
  console.log("User is active.");
}

Numbers

A number is a data type that represents a numeric value. In JavaScript, you can create a number using a numeric literal. This article will cover how to create and manipulate numbers, including how to perform arithmetic operations and how to use the Math object.

Example: Numeric Operations

let x = 10;
let y = 5;
let sum = x + y;
let product = x * y;
