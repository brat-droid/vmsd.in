Variables and Strings

Variables and strings are two of the most fundamental concepts in JavaScript. This article covers how to declare and use variables, and how to work with strings.

Variables

A variable is a container for a value. In JavaScript, you can declare a variable using the var, let, or const keyword. This article will cover the differences between these keywords and when to use each.

Example: Variable Declaration

let name = "Alice";
const age = 30;
var city = "New York";

Strings

A string is a sequence of characters. In JavaScript, you can create a string using single or double quotes. This article will cover how to create and manipulate strings, including how to concatenate strings, find the length of a string, and use template literals.

Example: String Manipulation

let greeting = `Hello, ${name}!`; // Template literal
let message = "Hello" + " World"; // Concatenation
let length = message.length;
