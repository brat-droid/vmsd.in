Objects

An object is a data structure that stores a collection of key-value pairs. This article covers how to create and manipulate objects in JavaScript.

Creating Objects

There are several ways to create an object in JavaScript, including using an object literal or the Object constructor. This article will cover the differences between these methods and when to use each.

Example: Object Literal

const person = {
  name: "Alice",
  age: 30,
  isStudent: false
};

Object Methods

JavaScript provides a number of built-in methods for working with objects. This article will cover some of the most common object methods, including how to add and remove properties, how to iterate over an object, and how to check if a property exists.

Example: Accessing Object Properties

console.log(person.name); // "Alice"
person.age = 31; // Update property
delete person.isStudent; // Delete property
