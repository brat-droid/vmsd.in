Recursion

Recursion is a programming technique in which a function calls itself. This article covers how to use recursion to solve problems in JavaScript.

Base Case and Recursive Step

A recursive function must have a base case, which is a condition that stops the recursion. It must also have a recursive step, which is where the function calls itself. This article will cover how to define the base case and the recursive step in a recursive function.

Example: Factorial Function (Recursive)

function factorial(n) {
  if (n === 0) { // Base case
    return 1;
  } else {
    return n * factorial(n - 1); // Recursive step
  }
}

When to Use Recursion

Recursion can be a powerful tool for solving certain types of problems, such as those that can be broken down into smaller, self-similar subproblems. This article will cover when to use recursion and when to use an iterative approach instead.
