Maps and Sets

Maps and sets are two new data structures that were introduced in ES6. This article covers how to use maps and sets in JavaScript.

Maps

A map is a collection of key-value pairs where the keys can be of any data type. This article will cover how to create and manipulate maps, including how to add and remove elements, how to iterate over a map, and how to get the size of a map.

Example: Using a Map

const myMap = new Map();
myMap.set("name", "Alice");
myMap.set(1, "one");
console.log(myMap.get("name")); // "Alice"

Sets

A set is a collection of unique values. This article will cover how to create and manipulate sets, including how to add and remove elements, how to iterate over a set, and how to perform set operations like union, intersection, and difference.

Example: Using a Set

const mySet = new Set();
mySet.add(1);
mySet.add(2);
mySet.add(1); // Ignored, as 1 is already in the set
console.log(mySet.has(2)); // true
