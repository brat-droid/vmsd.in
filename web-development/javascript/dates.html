Dates

The Date object in JavaScript is used to work with dates and times. This article covers how to create and manipulate dates in JavaScript.

Creating Dates

There are several ways to create a Date object in JavaScript, including using the Date constructor or by parsing a date string. This article will cover the different ways to create a Date object and when to use each.

Example: Creating a Date Object

const now = new Date();
const specificDate = new Date("2025-07-19T12:00:00");

Manipulating Dates

This article will cover how to get and set the different components of a date, such as the year, month, and day. We will also cover how to perform arithmetic operations on dates and how to format dates for display.

Example: Getting Date Components

const year = now.getFullYear();
const month = now.getMonth(); // 0-indexed
const day = now.getDate();
